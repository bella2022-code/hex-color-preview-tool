# Hex Color Preview Tool

A standalone browser tool for previewing one or many Hex colors, keeping local history, and saving individual colors or complete palettes.

## Features

- Parse `Color name #RRGGBB` lists in real time
- Preview multiple color swatches together
- Keep recent preview history in local storage
- Save individual colors or complete palettes
- Suggest familiar Chinese color names
- Rename, reload, and remove saved items

Open `index.html` directly or deploy the folder as a static site.
